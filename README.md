## Terminal Commands

1. Install NodeJs from [NodeJs Official Page](https://nodejs.org/en).
2. Go to your file project
3. To install package for server side: ```npm install```
4. To install package for client side: ```cd client``` then ```npm install```
5. To run concurrently client and server: ```npm run dev```
6. Navigate to `http://localhost:3000/`

+ Note: Server run on port 5000
